# PromptTrace

An end-to-end observability platform for LLM applications — trace, monitor, and analyze AI requests in real time.

## Overview

PromptTrace gives developers full visibility into how their LLM-powered applications behave in production. It captures every prompt, response, and associated metric so teams can debug failures, optimize cost, and enforce safety controls from a single dashboard.

## Features

- 🔍 **Request tracing** — Captures prompts, responses, latency, and token usage across every LLM call
- 📊 **Real-time dashboards** — Live visualization of model performance, throughput, and cost
- ⚡ **Battle-tested at scale** — Handles 10K+ interactions with stable query performance
- 🔐 **Security & governance** — Authentication and audit logging built in
- 🚦 **Rate limiting** — Configurable throttles to catch unsafe outputs and performance spikes
- 💰 **Cost analytics** — Per-model and per-user token spend breakdowns

## Tech Stack

- **Backend:** C#, ASP.NET Core
- **Frontend:** React
- **Database:** Azure Cosmos DB
- **Monitoring:** Azure Monitor
